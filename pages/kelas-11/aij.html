Administrasi Infrastruktur Jaringan - Fokus pada konfigurasi perangkat jaringan seperti Router dan Switch. Belajar VLAN, Routing, dan instalasi jaringan.

VLAN Routing Subnetting

Modul 1

VLAN Trunking

Memahami konsep VLAN dan cara menghubungkan antar switch menggunakan trunk.

Konsep VLAN

Pengertian Virtual LAN, keuntungan segmentasi jaringan, VLAN ID, dan cara kerja VLAN dalam switch.

25 menit Teori

Access & Trunk Port

Perbedaan port access dan trunk. Konfigurasi trunk untuk menghubungkan VLAN antar switch.

35 menit Praktik

Inter-VLAN Routing

Router on a stick dan Layer 3 switch untuk menghubungkan VLAN yang berbeda.

40 menit Praktik
Modul 2

Routing Statis

Konfigurasi routing secara manual untuk menghubungkan network yang berbeda.

Konsep Routing

Apa itu routing? Static vs Dynamic routing, routing table, dan cara kerja router dalam meneruskan paket data.

30 menit Teori

Konfigurasi Static Route

Konfigurasi static route pada Cisco dan MikroTik. Menghubungkan dua atau lebih network dengan route manual.

45 menit Praktik

Default Route & Gateway

Konfigurasi default gateway untuk mengarahkan traffic ke network yang tidak dikenal.

30 menit Praktik
Modul 3

Subnetting

Teknik pembagian jaringan untuk efisiensi pengalamatan IP.

Konsep Subnetting

Pengertian subnet, subnet mask, network address, broadcast address, dan range IP yang valid untuk host.

30 menit Teori

Perhitungan Subnet

Menghitung jumlah subnet, jumlah host per subnet, CIDR notation, dan konversi antar format subnet mask.

45 menit Praktik

VLSM & Supernetting

Variable Length Subnet Mask untuk efisiensi maksimal. Supernetting untuk menggabungkan network.

40 menit Praktik
