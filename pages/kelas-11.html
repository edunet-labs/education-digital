Tiga Spesialisasi TKJ

Masing-masing spesialisasi memiliki fokus yang berbeda. Pilih sesuai passion-mu!

AIJ

Administrasi Infrastruktur Jaringan

Fokus pada konfigurasi perangkat jaringan seperti Router dan Switch. Belajar VLAN, Routing, Subnetting, dan instalasi jaringan.

  • VLAN Trunking
  • Routing Statis
  • Instalasi Switch/Router
  • Konfigurasi Wireless
  • InterVLAN
Masuk AIJ

ASJ

Administrasi Sistem Jaringan

Fokus pada pengelolaan server berbasis Linux. Dari konfigurasi DHCP, DNS, hingga FTP Server untuk layanan jaringan.

  • Remote Server
  • DHCP Server
  • DNS Server
  • FTP Server
Masuk ASJ

KJ

Keamanan Jaringan

Fokus pada keamanan siber dan perlindungan infrastruktur IT. Belajar Firewall, Cyber Security, dan jenis ancaman online.

  • Firewall
  • Cyber Security
  • Tingkatan Cyber Security
  • Ancaman Siber
Masuk KJ
Tips Memilih

Bagaimana Memilih?

Pilih AIJ jika...

Kamu suka ngulik router dan switch, tertarik jadi Network Engineer, atau ingin kerja di ISP/provider.

Pilih ASJ jika...

Kamu suka Linux dan command line, ingin jadi System Admin, atau tertarik mengelola server perusahaan.

Pilih KJ jika...

Kamu tertarik cyber security, ingin jadi Security Analyst, atau suka tantangan mencari celah keamanan.
