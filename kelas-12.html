Pilih Jalurmu

Tiga Spesialisasi TKJ

Masing-masing spesialisasi memiliki fokus yang berbeda. Pilih sesuai passion-mu!

AIJ

Administrasi Infrastruktur Jaringan

Fokus pada konfigurasi perangkat jaringan seperti Router dan Switch. Belajar MikroTik, Cisco, hingga routing protocol tingkat lanjut.

  • RouterOS & MikroTik Configuration
  • Dynamic Routing (OSPF, BGP)
  • Advanced Firewall & NAT
  • Queue Tree & Traffic Control
Masuk AIJ

ASJ

Administrasi Sistem Jaringan

Fokus pada pengelolaan server berbasis Linux. Dari instalasi hingga konfigurasi berbagai layanan server yang dibutuhkan perusahaan.

  • Linux Server (Debian/Ubuntu)
  • Web Server & DNS Server
  • Mail Server & Database
  • Network Monitoring
Masuk ASJ

KJ

Keamanan Jaringan

Fokus pada keamanan siber dan perlindungan infrastruktur IT. Belajar ethical hacking, firewall, dan teknik hardening.

  • Cyber Security Fundamentals
  • Iptables & Firewall Rules
  • IDS/IPS & VPN
  • Server Hardening
Masuk KJ
Tips Memilih

Bagaimana Memilih?

Pilih AIJ jika...

Kamu suka ngulik router dan switch, tertarik jadi Network Engineer, atau ingin kerja di ISP/provider.

Pilih ASJ jika...

Kamu suka Linux dan command line, ingin jadi System Admin, atau tertarik mengelola server perusahaan.

Pilih KJ jika...

Kamu tertarik cyber security, ingin jadi Security Analyst, atau suka tantangan mencari celah keamanan.
